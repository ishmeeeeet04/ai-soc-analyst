# AI-Powered Autonomous SOC Analyst

An end-to-end AI system that ingests security logs, detects threats using machine learning,
explains its decisions with Explainable AI, maps findings to the MITRE ATT&CK framework,
and generates human-readable incident reports using an LLM + RAG pipeline.

## Status
🚧 In active development — Final Year Project

## Tech Stack
- Python, Flask (Backend)
- React (Frontend)
- Scikit-learn / XGBoost (ML)
- LLM + RAG (Incident summarization)
- MongoDB (Database)
